Git 常用指令

發表於 2024-05-13 分類於 超新手時期基礎筆記 閱讀次數：

.gitigonre 檔案會讓檔案不被 git 追蹤

Git command Description
git init 初始化 / 開始追蹤 git
git config –list 在 git 裡面的設定
git config –global http://user.name “name” 設定使用者
git config –global http://user.nameemail “name” 設定聯絡信箱
git status 目前狀態
git add . 新增所有檔案到 Staging Area
git add *.html 新增所有 html 檔案到 Staging Area
git commit -m “Creat : 新東東” 新增 commit & 他的訊息
git log git 日誌
git branch branchName 新增 branch 分支 ( Feature 新功能分支 )
git checkout branchName 切換 git branch 分支
git merge branchName 合併分支
git remote add origin gitSSH 增加遠端資料庫
git push -u origin master
設定完後，之後可以省略成 git push 從本地端推指定 branch 到 遠端
git remote remove origin 移除遠端資料庫的設定
git pull gitSSH 更新遠端新版本到本地端
git clone gitSSH 如過電腦上沒有就用 clone

git merge conflict 合併衝突
通常在 vscode 可以看衝突在哪裡發生
